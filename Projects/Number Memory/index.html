Number Memory

Explication du projet

Ce projet a pour objectif de développer un petit jeu de mémorisation de nombres en Java. L'utilisateur dispose de 10 secondes pour mémoriser une suite de chiffres qui s'affiche brièvement à l'écran. Il doit ensuite réécrire correctement la suite dans un champ de saisie. À chaque réussite, la longueur de la suite augmente de 1.


Objectifs

  • Utiliser l’IHM de Java (Swing) pour développer une application interactive.
  • Structurer une logique de jeu simple.
  • Appliquer des bonnes pratiques de développement en Java.
  • Apprendre à gérer des timers et des événements utilisateurs

Outils utilisés

  • Java
  • Swing

Version 1

Détection des cases et clics aléatoires. Aucun raisonnement logique, uniquement pour tester les interactions de base avec le jeu.

Version 2

Lecture de la grille avec des pixels precis dans chaque case pour identifié le nombre ou le type de case. Et ajout de logique, avec des condition certaine comme par exemple, si il ya le nombre 2 avec 2 cases autour cela ve dire qu'il y a 2 bombes donc il place 2 drapeau.

Version 3

Placement de drapeaux plus précis, meilleure gestion des probabilités lorsque les choix deviennent ambigus. Début d'une stratégie plus flexible.

Version 4

Optimisation du code, gestion de cas complexes, et meilleure stabilité. Le bot devient plus rapide et prend moins de mauvaises décisions.
